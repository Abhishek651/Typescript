# Statement

Build a console-based **Banking System**.

## Requirements

Implement the following features:

* Create a bank account
* Deposit money
* Withdraw money (with overdraft protection)
* Transfer money between accounts
* Display account details
* Display transaction history

## Rules

* Use TypeScript only.
* Do not use external libraries.
* Store data in memory (arrays/objects).
* Model accounts as a `class` implementing an `interface`.
* Apply the TypeScript concepts listed in the README wherever appropriate.

**Bonus**

* Apply an interest rate based on `AccountType`.
* Search accounts by owner name.
* Display total bank balance across all accounts.
