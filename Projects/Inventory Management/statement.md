# Project 3 - Inventory Management System

Build a console-based **Inventory Management System**.

## Requirements

Implement the following features:

* Add an inventory item
* Remove an inventory item
* Update inventory item information
* Update inventory item quantity
* Display all inventory items
* Calculate total inventory value
* Calculate average item price
* Prevent duplicate item IDs

## Rules

* Use TypeScript only.
* Do not use external libraries.
* Store data in memory (arrays and Records).
* Write clean, reusable functions.
* Apply the TypeScript concepts listed in the README wherever appropriate.
* Use dynamic data structures like `Record<string, Item>` for fast lookup.
* Implement `readonly` properties for immutable identifiers.
* Handle optional fields using `?` syntax and `Partial<T>` type.

**Bonus**

* Search items by name.
* Display the most valuable item.
* Filter items by price range.
