# Statement

Build a console-based **Student Management System**.

## Requirements

Implement the following features:

* Add a student
* Remove a student
* Update student information
* Update student marks
* Display all students
* Calculate the average marks of all students
* Prevent duplicate student IDs

## Rules

* Use TypeScript only.
* Do not use external libraries.
* Store data in memory (arrays).
* Write clean, reusable functions.
* Apply the TypeScript concepts listed in the README wherever appropriate.

**Bonus**

* Search students by name.
* Display the highest-scoring student.
* Sort students by marks.
