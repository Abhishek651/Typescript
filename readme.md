# TypeScript Practice

A structured collection of TypeScript projects to build practical skills through real-world challenges.

## Learning Path

| #  | Project              | Focus                                       |
| -- | -------------------- | ------------------------------------------- |
| 01 | Student Management   | Types, Interfaces, Functions, Arrays, Optional Parameters, `void`, Type Narrowing, CRUD |
| 02 | Banking System       | Classes, Enums, Error Handling              |
| 03 | Inventory Management | Utility Types, Dynamic Objects              |
| 04 | Generic Data Store   | Generics                                    |
| 05 | Shopping Cart        | Pick, Omit, `keyof`, `typeof`, `ReturnType` |
| 06 | Task Management      | Type Guards, Advanced Utilities             |
| 07 | Mini Backend         | Complete TypeScript Integration             |

## How to Use

1. Complete projects in order
2. Read the challenge statement
3. Build the solution
4. Revisit completed projects for revision

## Goal

Master modern TypeScript through hands-on projects, progressing from core language features to advanced type system concepts used in professional React and Node.js applications.
